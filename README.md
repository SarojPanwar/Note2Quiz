📘 Note2Quiz – AI-Powered Question Generator

Note2Quiz is an AI-driven exam question generation system that converts PDF or CSV lecture notes into exam-ready MCQs and Viva questions.
The application uses Natural Language Processing (NLP) and Large Language Models (LLMs) to generate questions strictly based on the uploaded content, ensuring academic relevance and accuracy.

This project is designed for students, educators, and exam preparation platforms.

🎯 Project Type

✅ Academic Project
✅ Group Project
✅ AI-Based Educational System
✅ Full-Stack Python Application (Streamlit-based)

✨ Features
📤 Upload PDF or CSV notes
🤖 AI-powered generation of:
📘 MCQs (Multiple Choice Questions)
🎤 Viva / Oral Exam Questions
🎚️ User-controlled:
Number of MCQs
Number of Viva questions
🧠 Automatic Bloom’s Taxonomy classification
📄 Export generated questions as Word (.docx) file
🎨 Clean and interactive Streamlit UI
🔐 Secure API key management using environment variables
📂 Input Formats Supported
✅ PDF files (.pdf)
✅ CSV files (.csv)

⚙️ How the System Works

User uploads a PDF or CSV file
System extracts and cleans the text
User selects:
Number of MCQs
Number of Viva questions
AI model analyzes the content
System generates:
✅ MCQs with correct answers
✅ Viva questions
✅ Bloom’s Taxonomy level
Results are displayed and can be downloaded as a Word document

🧠 AI & NLP Details

Uses Google Gemini API for intelligent question generation
Applies prompt engineering to ensure:
Exam-oriented questions
One correct answer per MCQ
Classifies questions using Bloom’s Taxonomy
Ensures all questions are generated only from the uploaded content
